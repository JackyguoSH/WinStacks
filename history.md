# WinStacks 变更日志

## [1.3] (2026-03-01) 代码质量与稳定性重大改进

### 新增功能
- 新增 `LogService` 日志服务类，提供统一的日志记录功能
- 新增 `ExceptionHandler` 异常处理服务类，提供安全的异常处理机制
- 新增 `AppConstants` 常量配置类，集中管理所有硬编码值
- 新增 `WindowsVersionService` Windows 版本检测服务类

### 改进
- 移除全局 Nullable 警告禁用，正确处理可空引用类型
- 完善全局异常处理器，添加日志记录和用户提示
- 实现 `IDisposable` 接口，正确释放资源（FileSystemWatcher、托盘图标等）
- 添加事件取消订阅方法，修复内存泄漏问题
- 使用常量类替换所有硬编码魔法数字
- 添加 Windows 版本兼容性检测，条件性应用 Windows 11 特性
- 所有方法添加 XML 文档注释，提升代码可读性

### 修复
- 修复资源泄漏问题（FileSystemWatcher 未正确释放）
- 修复内存泄漏问题（事件订阅未取消）
- 修复全局异常处理器形同虚设的问题
