# WinStacks (智能桌面叠放)

**Version 1.0 主要功能：**

1. **智能分类**：支持八种文件分类（照片、影片、音乐、压缩包、文档、应用程序、文件夹、其他）。
2. **自适应遮罩**：叠放展开界面会根据文件数量，自适应调整大小，防越界显示。
3. **沉浸式接管**：在启动/退出程序时，自动隐去/恢复原生桌面图标，做到无痕切换。
4. **可视化角标**：主图标会根据包含的文件动态显示层叠厚度，并在右上角提供文件总数角标。
5. **全量原生交互**：支持在展开界面单击高亮选中、双击打开、拖拽文件，以及呼出右键菜单。
6. **多阵营排版**：支持在托盘中随时更改图标排列风格（Win式左侧 / Mac式右侧 / 顶部水平 / 底部 Dock 式）。

## 如何使用

1. 前往页面右侧的 **Releases** 区域，下载最新的 `WinStacks.exe` 。
2. 双击运行。
3. 右击右下角系统托盘图标，可随时呼出控制面板切换排版，或安全退出。
