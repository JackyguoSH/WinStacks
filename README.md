# WinStacks (智能桌面叠放)

**WinStacks v1.3 主要功能：**

1. **智能分类**：支持八种文件分类（照片、影片、音乐、压缩包、文档、应用程序、文件夹、其他）。
2. **“最近使用”叠放**：新增特殊分类，实时聚合桌面最新修改过的文件，可自定义显示数量。
3. **自定义拖拽排版**：所有叠放均支持按住 `Shift` 键自由拖拽，支持随意摆放并具有智能边缘防撞与重叠避让机制。
4. **统一设置中心**：双击托盘图标呼出完整的设置面板，支持开机自启、一键隐藏桌面原生图标等功能。
5. **自适应遮罩**：叠放展开界面会根据文件数量，自适应调整大小，防越界显示。
6. **沉浸式接管**：在启动/退出程序时，自动隐去/恢复原生桌面图标，做到无痕切换。
7. **全量原生交互**：支持在展开界面单击高亮选中、双击打开、拖拽文件，以及呼出右键菜单。
8. **多阵营排版**：支持在托盘中随时更改图标排列风格（Win式左侧 / Mac式右侧 / 顶部竖向 / 底部水平式）。 
9. **中英双语**：全面支持中文与英文界面的无缝切换。

## 如何使用

1. 前往页面右侧的 **Releases** 区域，下载最新的 `WinStacks.exe` 。
2. 双击运行。
3. 右击右下角系统托盘图标，可随时呼出控制面板切换排版，或安全退出。
